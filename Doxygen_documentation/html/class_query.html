Detailed Description

A Query class which contains structured queries.

A Query object will be created by QueryParser after validation and parsing the user query. It will then be forwarded to QueryEvaluator for evaluation and to answer they query.

Query class contains the following relevant data:

vector<string> selectedSyn;
vector<bool> selectedSynIsCallProcedure;
string patternSyn;
vector<Relationship> relVect;
unordered_map<string, TypeTable::SynType> synTable;

It contains the declared synonyms, selected synonyms, Relationship vector, etc. It has public functions to retrieve the data mentioned above.
